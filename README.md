# Fictional Tour Company Website

This repository contains my final website project for the **HTML/CSS Responsive Web Design** class at Portland Community College. The project showcases a multi-page fictional tour 
company website built entirely with HTML and CSS, following mobile-first and responsive design principles.

## Features

- **Mobile-first design** for optimal viewing across devices  
- **Multi-page layout** including home, tours, about, and contact pages  
- **Responsive navigation** and content layout  
- Clean and semantic HTML structure  
- Styled entirely with CSS, demonstrating layout, typography, and visual hierarchy  

## Tools & Technologies

- **HTML5** – Structure and content of the website  
- **CSS3** – Styling, layout, and responsive design  

## Purpose

This project demonstrates my ability to:  
- Create fully responsive websites using HTML and CSS  
- Apply mobile-first design principles effectively  
- Organize multi-page website content in a clear, user-friendly way  

## Outcome

- Completed as a class project at Portland Community College  
- Earned a perfect score for the final assignment

**Note:** This website is a fictional project created for academic purposes. No JavaScript or backend functionality was used.
